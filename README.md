# Find My Notes

A simple note taking app.

### Setup:
```
npm install
```

### Start app:
```
npm run start
```

### Unit test:
```
npm run test
```

### e2e/feature test:
```
npm run selenium-setup   // run only once
npm run selenium-start   // app must also be started with npm start
npm run e2e-tests
npm run e2e-tests-watch  // if you want to run e2e in watch mode
```
